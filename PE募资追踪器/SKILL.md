# PE募资追踪器

追踪全球一级市场股权投资募资动态，重点关注中国和美国市场，自动生成LP情报报告。

**新增功能：上市公司CVC出资动态追踪** - 重点跟踪中国和美国上市公司出资私募股权基金的情况，分析产业资本投资趋势。

## 功能概览

- **每日简报**：每天早上8点推送最新募资动态
- **每周深度分析**：周日12点前完成汇总，输出重点LP跟进名单
- **重点区域**：中国、美国、欧洲
- **LP画像**：追踪重要LP的募资活动、投资偏好、决策动态
- **上市公司CVC追踪**：重点跟踪中美上市公司出资PE基金动态（**新增**）

## 定时任务设置

### 每日简报（8:00 AM）

```bash
openclaw cron add \
  --name "pe-daily-brief" \
  --schedule "0 8 * * *" \
  --task "搜索昨日全球PE募资动态，生成并发送简报"
```

### 每周汇总（周日12:00）

```bash
openclaw cron add \
  --name "pe-weekly-report" \
  --schedule "0 12 * * 0" \
  --task "生成本周PE募资汇总分析，输出重点LP跟进名单"
```

## 工作流程

### 每日简报流程

1. 搜索「过去24小时」PE募资新闻
2. 分类整理：
   - 中国GP募资
   - 美国GP募资
   - 欧洲/其他地区
   - LP出资动态
3. 生成结构化简报
4. 推送到指定渠道

### 每周汇总流程

1. 汇总本周所有募资信息
2. 深度分析：
   - 募资规模趋势
   - 重点LP活动分析
   - 行业/阶段分布
   - **上市公司CVC出资趋势（新增）**
3. 生成重点LP跟进名单：
   - 中国LP：主权基金、养老金、险资、家办
   - 美国LP：公共养老金、大学捐赠基金、母基金
   - 欧洲LP：欧洲投资银行、各国主权基金
   - **中国上市公司CVC：宁德时代、比亚迪、小米、药明康德等（新增）**
   - **美国上市公司CVC：Google、NVIDIA、Microsoft、Intel等（新增）**
4. 输出优先级建议（高/中/低）
5. **CVC战略协同分析（新增）**：分析被投企业与CVC母公司的业务协同机会
