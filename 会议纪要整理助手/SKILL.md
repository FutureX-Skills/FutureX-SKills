# 会议纪要整理助手

将录音转文字整理为结构化会议纪要

## 触发方式

上传录音转文字文件后，直接说：
- "整理成会议纪要"
- "生成详细纪要"
- "整理这份录音"
- "用纪要 skill 整理"

## 工作流程

### Step 1: 确认会议背景
我会询问：
- 会议主题/背景是什么？
- 参会人有哪些？（姓名、身份、角色）

### Step 2: 内容整理
根据录音转文字，我会：
1. 识别所有发言人
2. 按主题分章节
3. 去除语气词和重复内容
4. 保留所有核心信息（人名、机构、金额、时间、技术细节等）
5. 生成对话格式：`**姓名**：内容`

### Step 3: 输出格式

#### 标准输出包含：
1. **会议信息**（时间、地点、参会人）
2. **内容总结**（核心议题、关键结论、风险点）
3. **详细纪要**（按章节整理的对话）
4. **待办事项**（行动项清单）
5. **相关链接**（原始录音等）

#### 可选输出格式：
| 指令 | 输出 |
|------|------|
| `整理成会议纪要` | 完整详细版 |
| `提取金句` | 金句速查表（表格+引用） |
| `生成微信文章` | 可发布的推文格式 |
| `生成投研摘要` | 投资视角摘要（项目/团队/风险） |

### Step 4: 存档
自动生成飞书文档并保存，提供文档链接
