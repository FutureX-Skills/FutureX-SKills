# PPTX Logo & Label Fix

> 自动修复 PPTX 幻灯片中公司 Logo 与文字标签错位、遗漏、错乱的问题。

## 适用场景

- 幻灯片经过多次编辑后，Logo 下方的文字标签与实际公司不匹配
- 新增公司后遗漏了标签，或标签为空
- 重组幻灯片后标签顺序被打乱
- 需要为英文公司名补充中文分类标签
- 任何涉及"Logo 和标签对不上"的任务

适用于 FutureX 投资组合幻灯片及任何其他 PPTX 格式的 portfolio/map 页面。

---

## 核心能力

### 1. 智能提取
从 PPTX（ZIP 压缩包）中解析出每张幻灯片里的所有 Logo 分组，按左右/上下顺序排列，提取出每个公司的 logo 图片供参考识别。

### 2. 视觉识别
通过读取 Logo 图片，结合 `references/futureX-companies.md` 内置的公司列表进行匹配；列表外的公司会通过视觉识别 + 网络搜索确认身份，并推荐合适的中文分类标签。

### 3. 安全编辑
直接对 PPTX XML 做字符串替换修改，**绝不**使用 ElementTree 等 XML 解析器重序列化（会导致命名空间损坏、图片丢失、静默腐败文件）。

### 4. 版本管理
修复后的文件自动递增版本号（如 `v2.pptx` → `v3.pptx`），原始文件完整保留。

---

## 工作流程

```
上传 PPTX
    ↓
Step 1: 解压并加载所有文件
    ↓
Step 2: 提取所有 Logo 分组（位置 + 当前文字 + 图片）
    ↓
Step 3: 视觉识别每家公司 → 确定正确标签
    ↓
Step 4: 字符串替换 XML，写入新标签
    ↓
Step 5: 重建 PPTX，验证完整性
```

## 常见问题

**输出文件打开后图片/格式损坏** → 检测到 XML 命名空间被破坏（`ns2:` 前缀），需重新执行，过程中不可使用 XML 解析器。

**shape ID 找不到对应公司** → 使用 `shape-id-discovery.md` 中的脚本重新扫描当前版本幻灯片的 group 信息。

**无法识别某个 Logo** → 输出 Logo 文件名和坐标，直接询问用户确认。
