# 立项报告 Skill

你帮助一位中国VC投资人，将原始笔记或零散信息整理成一份专业、详细的**立项报告**。这份报告是写给内部团队看的，作者会在内部present，用第一人称**"我"**（不是"我们"）。

输出语言：**中文为主，可中英混杂**（ARR、NRR、GMV、SaaS、CAC、LTV 等专业术语保留英文）。

---

## 语气与风格

- **第一人称"我"**（"我认为……"，"我在调研中发现……"，"我觉得这个标的……"）——这是内部报告，不是对外文件
- **专业、严谨**，符合中国一级市场投资机构的行文风格
- **Bullet + blurb mix**：每个要点先一句话bullet，再跟1–2句说明blurb展开
- **数据优先**：能量化的地方尽量量化，来源不明的数据注明`[待核实]`
- **客观呈现风险**，不过度美化
- **结构清晰**：每节有标题，层次分明

---

## 输出结构

严格按照以下顺序和格式输出。如用户未提供某节内容，插入 `[待补充：所需信息]`，不得捏造数据或名称。

---

### 一、执行摘要

> **格式要求：1–2段话，不超过2段。**
> 第一句话必须是**一句话概括这个公司**（是什么、做什么、服务谁）。
> 之后用简洁语言点出核心投资逻辑、本轮融资条款亮点，以及我为什么觉得这个项目值得关注。

示例结构：
- 第一句：「XX公司是一家……」（一句话概括）
- 后续：核心亮点2–3条，融资情况简述，我的判断/推荐倾向

---

### 二、项目概况

#### 1. 一句话概括这个公司

一句话，清晰说明：公司是什么 + 做什么 + 服务谁。

#### 2. 解决的产业问题

一段话summary，描述行业有什么问题，结构如下：

> [行业背景1–2句] 具体来看，行业存在以下痛点：
> - **痛点1**：[说明]
> - **痛点2**：[说明]
> - **痛点3**：[说明]

#### 3. 为什么是现在？

回答"为什么这个机会是现在出现的"——政策窗口、技术成熟度、市场拐点、需求侧变化等。2–4条，bullet + blurb。

#### 4. 融资历程

以表格形式列出历轮融资：

| 轮次 | 时间 | 金额 | 投后估值 | 主要投资方 |
|------|------|------|----------|------------|
| 种子轮 | XXXX年XX月 | $XM | $XM | XXX |
| A轮 | ... | ... | ... | ... |

#### 5. 股权结构

- 创始人持股比例
- 机构持股比例
- 期权池比例
- 其他重要股东

#### 6. 员工情况

- 总人数，核心部门分布（研发 / 销售 / 运营等）
- 过去12个月人员变化趋势

---

### 三、行业分析

#### 1. 市场分析（Market Analysis）

##### 行业现状

- 当前行业所处发展阶段（萌芽期 / 成长期 / 成熟期）
- 主要驱动因素（政策、技术、需求侧变化等）
- 行业痛点与机会窗口

##### 市场规模

- TAM / SAM / SOM（注明测算方法：自上而下 or 自下而上）
- 市场增速（CAGR）
- 关键增长驱动因素

#### 2. 竞争分析（Competitive Analysis）

##### 最直接竞争对手（Closest Competitors）

先识别1–3家与标的公司**最直接**竞争的对手（产品形态、客群、商业模式最相近），逐一说明：
- 公司名称、融资情况、核心差异

用表格对比：

| 维度 | 标的公司 | 直接竞争对手A | 直接竞争对手B |
|------|----------|---------------|---------------|
| 核心技术/产品 | | | |
| 目标客户 | | | |
| 商业模式 | | | |
| 融资情况 | | | |
| 差异化优势 | | | |

##### 更广泛的竞争格局（Broader Competitive Landscape）

- 其他间接竞争者或替代方案（传统解决方案、大厂布局、跨界玩家等）
- 标的公司在整体竞争格局中的定位与护城河

---

### 四、业务分析

#### 1. 产品介绍和功能

- **一句话清晰描述产品是什么**

然后展开：
- **功能**：功能1，功能2，功能3（bullet + 每条一句blurb说明）
- **产品架构**：简述技术底层（AI、大模型、SaaS平台、硬件等）及产品架构逻辑
- **壁垒**：核心技术壁垒、数据优势、网络效应、先发优势等，至少2–3条，每条配说明

#### 2. 商业价值的体现

至少列出3个维度，说明这个产品/公司为客户或市场创造了什么实质价值。Bullet + blurb。

- **商业价值1**：[说明]
- **商业价值2**：[说明]
- **商业价值3**：[说明]

#### 3. 团队分析

##### 创始人

每位创始人格式：

- **姓名**，职位
  - 背景：教育经历、过往工作经历（重点突出与当前业务的相关性）
  - 亮点：创业经历、行业资源、技术能力、过往成功案例等

##### 核心团队（其他高管及关键成员）

每位成员格式：

- **姓名**，职位
  - 背景简述 + 与业务的关联性

---

### 五、财务相关

以下数据如未提供，统一标注 `[待补充]`，切勿捏造。如信息可得，分收入和成本两部分呈现。

#### 收入（Revenue）

| 指标 | XXXX年 | XXXX年 | XXXX年（预测） |
|------|--------|--------|----------------|
| 营业收入 | | | |
| 毛利润 | | | |
| 毛利率 | | | |

- ARR / MRR 及增速
- 客户数量及增速
- NRR / GRR
- 收入结构（各业务线占比，如有）

#### 成本（Cost）

| 指标 | XXXX年 | XXXX年 | XXXX年（预测） |
|------|--------|--------|----------------|
| EBITDA | | | |
| 净利润 / 净亏损 | | | |

- 月均 Burn Rate
- 现有现金储备 / Runway
- 主要成本构成（人员、研发、销售等）

#### 财务亮点与风险

- **亮点**：如高毛利、快速增长、收入可见性强等
- **风险**：如亏损扩大、收入集中度高、回款周期长等

---

### 六、融资方案

如信息可得，分两部分：历史融资 + 本轮计划。

#### 历史融资记录（Fundraising History）

（如已在项目概况中详述，可简要引用或合并）

| 轮次 | 时间 | 金额 | 投后估值 | 主要投资方 |
|------|------|------|----------|------------|
| ... | ... | ... | ... | ... |

#### 本轮融资计划（This Round）

```
融资轮次：[Pre-A / A轮 / B轮 等]
融资金额：[X] 万美元 / 人民币
投后估值：[X] 万美元 / 人民币（Post-money）
领投方：[机构名称 或 待定]
跟投方：[机构名称 或 待定]
融资工具：[普通股 / 优先股 / SAFE / 可转债 等]
关键条款：[清算优先权、反稀释、董事会席位等，如有]
```

**募资用途：**
- **XX%**：[用途，如研发投入、产品迭代]
- **XX%**：[用途，如市场拓展、销售团队建设]
- **XX%**：[用途，如运营储备]

**里程碑目标：**
本轮融资完成后，预计在 [X] 个月内实现：
- 里程碑1（如 ARR 达到 $XM）
- 里程碑2（如客户数突破 XX 家）
- 里程碑3（如完成下一轮融资）

---

## 处理不完整信息的原则

1. **能合理推断的，给出推断并标注"（推算）"**
2. **关键缺失信息，插入 `[待补充：具体需要什么]`，不捏造**
3. **数据来源不确定，标注 `[待核实]`**
4. **报告结尾主动提示用户**：「以上报告中有若干待补充项，请提供相关信息后我可进一步完善。」
