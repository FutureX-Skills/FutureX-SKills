# 飞书生日同步

## 功能概述

将 Excel 格式的生日列表**增量同步**到飞书日历，自动：
- 检测已有事件，避免重复创建
- 生成随机生日祝福文案
- 创建年度重复的生日提醒

## 支持的 Excel 格式

**必需列：**
- `姓名` 或 `姓名（必填）` - 人员姓名
- `生日` - 生日日期，支持多种格式

**支持的日期格式：**
- 标准日期：`2021-05-26`、`2021/05/26`
- 中文格式：`12月16日`、`3月5日`

## 工作流程

### 1. 读取 Excel 数据

解析 Excel 生成事件列表：

```bash
python3 scripts/read_excel.py /path/to/birthday.xlsx
```

### 2. 增量同步到飞书日历

**特点：**
- ✅ 只创建新事件，跳过已存在的
- ✅ 自动生成生日祝福文案
- ✅ 全天事件 + 当天弹窗提醒

```bash
# 预览模式（查看将创建哪些事件）
python3 scripts/sync_calendar.py /path/to/birthday.xlsx --dry-run

# 实际同步（当前年份）
python3 scripts/sync_calendar.py /path/to/birthday.xlsx

# 同步指定年份
python3 scripts/sync_calendar.py /path/to/birthday.xlsx --year 2026
```

## 事件设置

| 属性 | 设置 |
|------|------|
| 标题 | 🎂 {姓名} 生日 |
| 提醒 | 当天早上弹窗通知 |
| 重复 | 每年自动重复 |
| 描述 | 随机生日祝福文案 |
| 日历 | 飞书默认日历 |

## 生日祝福文案示例

每次创建事件时，会从以下文案中随机选择一条：

- "愿你的每一天都如生日般美好，岁岁平安，年年如意！"
- "新的一岁，愿你遇见更好的自己，收获更多的快乐与惊喜！"
- "生日快乐！愿你眼中有光，心中有爱，未来可期！"
- ...（共 10 条，可在脚本中自定义）

## 增量同步逻辑

1. 查询指定年份已有的生日事件
2. 对比 Excel 数据，找出**尚未创建**的生日
3. 只为新人员/新日期创建事件
4. 已有事件完全不受影响

**适用场景：**
- 新员工入职，追加到已有生日列表
- 每年更新，不会重复创建去年的事件

## 注意事项

- 2月29日的生日在非闰年会自动调整为2月28日
- 事件创建后会显示在飞书日历的"全天事件"区域
- 如需删除或修改，直接在飞书日历中操作即可
