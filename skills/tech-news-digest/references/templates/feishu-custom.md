# AI热点日报 - Feishu定制模板

## 格式要求

1. **分类标题** - 使用大号字体（**加粗**）
2. **每则信息** - 包含：标题 + 链接 + 2-3句话要点介绍
3. **结尾精选** - 挑选2个最有价值的内容，提出2个讨论话题

## 生成规则

1. **数据收集**：运行 pipeline 获取最新资讯
2. **排序**：每个分类内按 quality_score 降序排列
3. **摘要生成**：
   - 如果有 full_text，从中提取要点
   - 如果没有，基于 title + snippet 写 2-3 句话总结
4. **精选挑选**：
   - 选择 quality_score 最高的2条
   - 或选择最具市场/行业影响力的2条
5. **讨论话题**：
   - 基于精选内容的行业影响、技术趋势、商业机会等角度提问
   - 问题要有开放性，能引发深入讨论

## 输出要求

- 使用中文
- 标题加粗加大（使用 **）
- 每则信息之间用空行分隔
- 链接单独一行
- 精选部分必须有深度，不只是重复标题
